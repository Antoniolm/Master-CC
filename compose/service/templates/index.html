Patch principales del servicio

  • /object3d
    • GET -> Obtiene todos los objetos 3d del sistema
    • POST -> Crea un nuevo objeto 3d en el sistema
  • /object3d/fileName
    • GET -> Obtiene el objetos 3d con el nombre especificado
    • POST -> Actualiza la información de un objetos 3d
    • DELETE -> elimina el objeto 3d con el nombre especificado
  • /status
    • Devuelve el estado del servicio
